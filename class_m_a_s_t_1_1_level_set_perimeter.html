void MAST::LevelSetPerimeter::evaluate ( )
virtual

this is the abstract interface to be implemented by derived classes.

This method calculates the quantity . This is provided as an interface since some quantities requires evaluation of all inputs before the final quantity can be calculated. So, the user can call this routine on each element before requesting the output values using either output_for_elem() or output_total().
